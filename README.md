# Posts API (Node.js + Express + Supabase)

A RESTful API for managing posts with **JWT authentication** and **ownership-based authorization**.

---

# Features

✅ User authentication (Register / Login)  
✅ JWT-based authorization  
✅ Full CRUD for posts  
✅ Protected routes (middleware)  
✅ Ownership control (users can only modify their own posts)  
✅ Structured backend architecture  

---

# Tech Stack

- Node.js
- Express.js
- Supabase (PostgreSQL)
- JWT (`jsonwebtoken`)
- bcrypt (password hashing)
